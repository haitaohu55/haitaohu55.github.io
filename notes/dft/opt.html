Structure Optimization using VASP

Last updated:
Contents

    1. Basic idea

    Structure optimization aims to find atomic positions $\mathbf{R}_i$ that minimize the total energy $E(\mathbf{R}_i)$ or make all forces smaller than a given threshold, e.g. $|F_i| \lt 0.01 \text{ eV/Å}$.

    In VASP this is usually done by setting $\texttt{IBRION}$, $\texttt{ISIF}$ and $\texttt{NSW}$ properly so that the ions are allowed to relax.

    2. Typical VASP input files

    2.1 INCAR (toy example)

    SYSTEM = Example relaxation
ENCUT  = 520
EDIFF  = 1E-6
EDIFFG = -0.02     ! force criterion (eV/Å)

IBRION = 2         ! conjugate-gradient relaxation
ISIF   = 2         ! relax ions, keep cell fixed
NSW    = 80        ! max ionic steps

ISMEAR = 0
SIGMA  = 0.05

    2.2 KPOINTS & POTCAR

    Choose a reasonably K-space path，for example $9 × 9 × 1$ for 2D systems

    $\texttt{POTCAR}$ can be generated using vaspkit.

    3. Output files

    The file $\texttt{CONTCAR}$ is what you need, and $\texttt{OUTCAR}$ contains detailed information about the relaxation process.
