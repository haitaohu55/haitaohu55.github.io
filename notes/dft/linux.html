    1. 文件管理

    
pwd               # 查看当前位置

# 列出文件
ls                # 列出所有文件
ls -lh OUTCAR     # 列出 OUTCAR 文件大小

# 切换目录
cd ~/project_hu   # 进入某目录 project_hu
cd                # 回到 home
cd ..             # 回到上一级

# 新建目录
mkdir 1_opt        # 建目录 1_opt
mkdir -p 1_opt/OUTPUT  # 一次建多级目录

# 复制
cp POSCAR POSCAR1   # 复制 POSCAR 为 POSCAR1
cp WAVECAR ../2_band # 复制到目录
cp -v 1_scf/{WAVECAR,POTCAR,CHGCAR} 2_band/  # 复制多个文件
mv CONTCAR POSCAR   # 重命名

    2. 查看与搜索文件内容

    
# 快速查看
cat OUTCAR          # 一次性打印
head OUTCAR         # 前 10 行
tail OUTCAR         # 后 10 行

# 在文件中搜索
grep fermi OUTCAR           # 找到含 fermi 的行

    3. 简单编辑：vim / vi

    
vi INCAR     # 或 vim INCAR

# 进入 vim 后常用操作：
i           # 进入插入模式(可以打字)
Esc         # 退出插入模式
:w          # 保存
:q          # 退出
:wq         # 保存并退出
:q!         # 强制退出(不保存)

    4. Slurm 作业管理

    
# 提交作业
sbatch job.slurm

# 查看作业
squeue     # 查看自己所有任务

# 取消作业
scancel 123456

    5. GitHub 基本操作

    用于同步本地个人主页 / 笔记到 GitHub Pages。 

    
# 进入本地目录
cd "/Users/huhaitao/Desktop/个人主页"

# 暂存修改
git add index.html    # 暂存 index.html 的修改    
git add .             # 把所有修改都暂存

# 本地提交
git commit -m "some message"  # 提交暂存的修改，附加说明信息

# 与 GitHub 同步
git pull origin main       # 先拉远程(有需要时)
git push origin main       # 再推送本地修改

    6. 远程登录与传文件：SSH

    从本地连接到集群，以及在本地和集群之间传输文件。 

    
# 登录集群
ssh haitaohu@192.168.10.53
